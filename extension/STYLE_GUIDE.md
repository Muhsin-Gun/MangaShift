# MangaShift Style Guide

## smooth
- Goal: polished anime output with soft gradients and clean edges.
- Line weight: medium, anti-aliased.
- Color direction: bright but balanced.

## shonen
- Goal: high-energy action style with punchy contrast.
- Line weight: bold and sharp.
- Color direction: high saturation, dramatic highlights.

## shoujo
- Goal: emotional, gentle visual tone.
- Line weight: delicate and smooth.
- Color direction: pastel and warm highlights.

## realistic
- Goal: cinematic realism while preserving composition.
- Line weight: reduced stylization.
- Color direction: natural skin/lighting, cooler shadows.

## gritty
- Goal: dark and rough comic atmosphere.
- Line weight: heavier and textured.
- Color direction: desaturated with strong black levels.

## painterly
- Goal: brushstroke-like artistic rendering.
- Line weight: softened into painted strokes.
- Color direction: rich and blended.
